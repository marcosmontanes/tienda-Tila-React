# tienda-tila

Tienda online en React JS

Este repositorio contiene una aplicación creada con React + Vite. Conservé parte de la información del template original para referencia y para facilitar el desarrollo.

Breve nota técnica:

- Plantilla: React + Vite
- Plugins útiles: `@vitejs/plugin-react` o `@vitejs/plugin-react-swc`

Para iniciar el proyecto localmente:

1. Instala dependencias:

	npm install

2. Ejecuta en modo desarrollo:

	npm run dev

Más detalles y recomendaciones están en el `package.json`.
